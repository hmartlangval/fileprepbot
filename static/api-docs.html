Bot Management API Documentation

This API allows you to manage bot instances programmatically. All endpoints return JSON responses.

Back to Web Interface

GET/api/bots

Get a list of all active bots.

Response

{ "bots": [ { "id": "base_bot", "name": "Base Bot Demo", "type": "base", "channel": "general", "connected": true }, { "id": "custom_bot", "name": "Custom Bot Demo", "type": "custom", "channel": "general", "connected": true } ] }

GET/api/bot-instance-types

Get a list of available bot types.

Response

{ "types": ["base", "custom", "test"] }

POST/api/bots

Create a new bot instance.

Request

{ "bot_id": "my_bot", // Optional, auto-generated if not provided "bot_name": "My Bot", // Required "channel": "general", // Optional, defaults to "general" "bot_instance_type": "base", // Optional, defaults to "base" "bot_type": "task_bot", // Optional, defaults to "task_bot" // Additional options based on bot type: "persona": "friendly" // Only for "custom" bot type }

Response

// Success { "success": true, "bot_id": "my_bot" } // Error { "success": false, "error": "Could not create bot of type xyz" }

DELETE/api/bots/{bot_id}

Stop a specific bot instance.

Response

{ "success": true }

DELETE/api/bots

Stop all active bot instances.

Response

{ "success": true }

POST/api/bots/{bot_id}/command

Send a command to a specific bot instance.

Request

{ "command": "/join general" }

Response

// Success { "success": true } // Error { "success": false, "error": "No command provided" }
